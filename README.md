# Marcelo Javier Fernandez Caballero

Prompt engineering, AI generativa e formazione professionale.

Creo strumenti pratici per trasformare l'AI in lavoro reale: formazione, ristorazione, design, siti web, sicurezza, contenuti digitali, musica generativa e automazione su Mac.

## Progetto principale

### Marcelus.v.v01

Framework italiano di prompt engineering professionale con **1151 prompt** pronti all'uso.

Include prompt per:

- design e identita visiva;
- siti web e contenuti digitali;
- formazione professionale;
- educazione;
- sicurezza;
- informazioni di quartiere;
- album musicali con SUNO;
- programmazione, risk management, code creation, antiriciclaggio, sicurezza aziendale e Terminal Mac;
- spesa familiare, consumi, convenienza e tecniche di acquisto;
- analisi finanziarie e banche;
- F&B tradizionale, turistico, spiaggia e servizio all ombrellone;
- contabilita ristorativa, andamento bar, proiezioni e miglioramento gestionale;
- bar in periferia romana, Gratta e Vinci e stagionalita;
- politica, partiti e analisi neutrale;
- GitHub pratico, portfolio e monetizzazione;
- analisi di tesi universitaria.

[Apri Marcelus.v.v01](https://github.com/manzo8b/marcelus-v-v01)

## Metodo di lavoro

- Prompt strutturati, non improvvisati
- Output pratici e pronti da usare
- Privacy, anti-allucinazione e controllo delle fonti
- Linguaggio professionale ma accessibile
- Applicazione concreta in contesti italiani ed europei
